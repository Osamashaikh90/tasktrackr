# Tasktrackr

- TaskTrackr is a todo-list app using NextJs with TS and Employed PostgreSQL as the database solution.

## Technologies Used 🛠️
### Frontend:
+ Next.js
+ TypeScript
+ React

### Backend:
+ Node.js
+ Express.js
+ PostgreSQL (Database)
